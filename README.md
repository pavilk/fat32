# FAT32 Image Tool

Проект представляет собой CLI и GUI утилиту для работы с образом файловой системы **FAT32** без его монтирования в ОС.

Поддерживается просмотр, чтение, копирование, добавление, удаление и переименование файлов и каталогов внутри FAT32-образа.

---

## Возможности

### CLI (fattool.py)

Поддерживаемые команды:

- `ls [PATH]` — список файлов и каталогов
- `cat PATH` — вывести содержимое файла
- `put SRC DST [-r]` — скопировать файл/каталог с хоста в образ
- `get SRC DST` — скопировать файл из образа на хост
- `rm PATH` — удалить файл или каталог (рекурсивно)
- `mv SRC DST` — переименовать или переместить файл/каталог

Флаги:
- `-i IMAGE` — путь к FAT32-образу (обязательно)
- `-r` — рекурсивная операция (для каталогов)

Примеры:
```bash
python fattool.py -i fat32.img ls /
python fattool.py -i fat32.img put ./folder /folder -r
python fattool.py -i fat32.img rm /folder
python fattool.py -i fat32.img mv /old.txt /new.txt
```

---

### GUI

Графический интерфейс реализован на **Tkinter**.

Возможности GUI:
- открытие FAT32-образа
- навигация по каталогам
- просмотр файлов
- удаление файлов и каталогов
- переименование (через move)
- отображение имён, типа и размера файлов

GUI расположен в:
```
fat32/gui/app.py
```

Запуск:
```bash
python fattool.py --gui
```
